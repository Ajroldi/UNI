## Progetto 1: Conversione di Temperature (Fahrenheit ‚Üí Celsius)

### Obiettivi di Apprendimento
Questo primo progetto introduce i concetti fondamentali di Python attraverso un problema pratico:
- Input da riga di comando con `input()`

- Assegnazione di variabili

- Output con `print()`
- Definizione di funzioni
- Importanza dell'indentazione in Python

### Il Problema
Vogliamo creare un programma che converte una temperatura da gradi Fahrenheit a gradi Celsius usando la formula:

$$
¬∞C = \frac{5}{9}(¬∞F - 32)
$$

### Approccio 1: Implementazione Diretta (senza funzioni)

```python
# Richiediamo all'utente di inserire la temperatura in Fahrenheit
fahrenheit_string = input("Inserisci la temperatura in Fahrenheit: ")

# Convertiamo la stringa in un numero decimale
fahrenheit = float(fahrenheit_string)

# Applichiamo la formula di conversione
celsius = (fahrenheit - 32) * 5/9

# Mostriamo il risultato
print(celsius)
```

**Spiegazione passo-passo:**
1. `input()` restituisce sempre una **stringa** (tipo `str`), anche se l'utente digita dei numeri
2. `float()` converte la stringa in un numero decimale (tipo `float`) necessario per i calcoli
3. L'operazione matematica segue l'ordine standard delle operazioni (PEMDAS)
4. `print()` visualizza il risultato a schermo

**Limitazioni di questo approccio:**
- Se vogliamo convertire pi√π temperature, dobbiamo riscrivere o rieseguire tutto il codice
- Il codice non √® riutilizzabile in altre parti del programma
- Difficile da testare e mantenere

### Approccio 2: Implementazione con Funzioni (metodo preferito)

```python
def convert(s):
    """
    Converte una temperatura da Fahrenheit a Celsius.
    
    INPUT:
        s - stringa contenente la temperatura in Fahrenheit
    OUTPUT:
        temperatura convertita in Celsius (float)
    """
    f = float(s)
    c = (f - 32) * 5/9
    return c

# Esempio di utilizzo
temperatura_input = input("Inserisci la temperatura in Fahrenheit: ")
risultato = convert(temperatura_input)
print(f"La temperatura in Celsius √®: {risultato}¬∞C")
```

**Vantaggi dell'uso di funzioni:**
1. **Riutilizzabilit√†:** Possiamo chiamare `convert()` quante volte vogliamo senza riscrivere il codice
2. **Modularit√†:** La logica di conversione √® isolata e facile da testare
3. **Leggibilit√†:** Il codice diventa pi√π chiaro e auto-documentato
4. **Manutenibilit√†:** Se la formula cambia, modifichiamo solo un punto

**Elementi chiave della sintassi Python:**
- `def` √® la parola chiave per definire una funzione
- L'**indentazione** (4 spazi o 1 tab) √® obbligatoria e definisce il blocco di codice della funzione
- La **docstring** (testo tra `"""..."""`) documenta la funzione ed √® accessibile con `help(convert)`
- `return` specifica cosa restituisce la funzione

## Progetto 2: Equazione di van der Waals

### Obiettivi di Apprendimento
Questo progetto amplia le nostre competenze introducendo:
- Funzioni con **parametri multipli**
- Uso della funzione `help()` per consultare la documentazione
- **Formattazione delle stringhe** (notazione scientifica)
- **Argomenti opzionali** con valori predefiniti
- Differenza tra **argomenti posizionali** e **argomenti per parola chiave**

### Il Contesto Fisico

L'**equazione di stato di van der Waals** √® una modifica dell'equazione dei gas ideali che tiene conto delle interazioni molecolari e del volume occupato dalle molecole stesse. √à particolarmente importante in termodinamica e fisica statistica.

**Formula:**
$$
\left[P + a\left(\frac{n}{V}\right)^2\right](V - nb) = nRT
$$

**Variabili:**
- $P$ = pressione [Pa]
- $V$ = volume [m¬≥]
- $T$ = temperatura [K]
- $n$ = numero di moli
- $R$ = costante universale dei gas = 8.314 J/(mol¬∑K)
- $a$ = costante di van der Waals specifica del gas [Pa¬∑m‚Å∂/mol¬≤] (rappresenta le forze attrattive)
- $b$ = costante di van der Waals specifica del gas [m¬≥/mol] (rappresenta il volume molecolare)

**Costanti di van der Waals per diversi gas:**

| Gas           | a [Pa¬∑m‚Å∂/mol¬≤] | b [m¬≥/mol]   |
|---------------|----------------|--------------|
| Elio (He)     | 3.46 √ó 10‚Åª¬≥    | 23.71 √ó 10‚Åª‚Å∂ |
| Neon (Ne)     | 2.12 √ó 10‚Åª¬≤    | 17.10 √ó 10‚Åª‚Å∂ |
| Idrogeno (H‚ÇÇ) | 2.45 √ó 10‚Åª¬≤    | 26.61 √ó 10‚Åª‚Å∂ |
| CO‚ÇÇ           | 3.96 √ó 10‚Åª¬π    | 42.69 √ó 10‚Åª‚Å∂ |
| Vapore acqueo | 5.47 √ó 10‚Åª¬π    | 30.52 √ó 10‚Åª‚Å∂ |
| Ossigeno (O‚ÇÇ) | 1.38 √ó 10‚Åª¬π    | 31.86 √ó 10‚Åª‚Å∂ |

### Problema da Risolvere
Dato un gas con parametri $T$, $V$, $n$, $a$ e $b$, calcolare la pressione $P$ risolvendo l'equazione per $P$:

$$
P = \frac{nRT}{V - nb} - a\left(\frac{n}{V}\right)^2
$$

### Implementazione Versione 1: Funzione Base

```python
def vanderwaals(T, V, n, a, b):
    """
    Calcola la pressione usando l'equazione di van der Waals.
    
    INPUT:
        T - temperatura [K]
        V - volume [m¬≥]
        n - numero di moli
        a - costante di van der Waals [Pa¬∑m‚Å∂/mol¬≤]
        b - costante di van der Waals [m¬≥/mol]
    OUTPUT:
        P - pressione [Pa]
    """
    R = 8.314  # costante universale dei gas [J/(mol¬∑K)]
    P = n * R * T / (V - n * b) - a * n**2 / V**2
    return P

# Esempio: 1 mole di ossigeno in 0.001 m¬≥ a 273.15 K
P = vanderwaals(273.15, 0.001, 1, 1.38e-1, 31.86e-6)
print(f"Pressione: {P} Pa")
```

**Note sull'operatore di potenza:**
- In Python, `**` √® l'operatore di elevamento a potenza
- `n**2` significa "n al quadrato"
- `V**2` significa "V al quadrato"

### Formattazione dell'Output: Notazione Scientifica

I numeri molto grandi o molto piccoli sono pi√π leggibili in notazione scientifica:

```python
# Formattazione in notazione scientifica
P_sn = "{:e}".format(P)
print(f"Pressione in notazione scientifica: {P_sn}")

# Oppure con f-string (Python 3.6+)
```


### Implementazione Versione 2: Parametri Predefiniti

Spesso lavoriamo con lo stesso gas. Invece di specificare sempre le stesse costanti, possiamo usare **valori predefiniti**:

```python
def vanderwaals1(T, V, n, a=1.38e-1, b=31.86e-6):
    """
    Calcola la pressione usando l'equazione di van der Waals.
    Valori predefiniti: ossigeno (O‚ÇÇ)
    
    INPUT:
        T - temperatura [K]
        V - volume [m¬≥]
        n - numero di moli
        a - costante di van der Waals [Pa¬∑m‚Å∂/mol¬≤] (default: O‚ÇÇ)
        b - costante di van der Waals [m¬≥/mol] (default: O‚ÇÇ)
    OUTPUT:
        P - pressione [Pa]
    """
    R = 8.314
    P = n * R * T / (V - n * b) - a * n**2 / V**2
    return P
```

### Modalit√† di Chiamata: Argomenti Posizionali vs. per Parola Chiave

Python offre grande flessibilit√† nel chiamare funzioni con parametri opzionali:

```python
# 1. Usa valori predefiniti (ossigeno)
P1 = vanderwaals1(273.15, 0.001, 1)

# 2. Specifica solo 'a', usa default per 'b'
P2 = vanderwaals1(273.15, 0.001, 1, a=3.46e-3)

# 3. Specifica 'a' e 'b' in ordine posizionale
P3 = vanderwaals1(273.15, 0.001, 1, 3.46e-3, 23.71e-6)

# 4. Specifica 'a' e 'b' per parola chiave (ordine qualsiasi)
P4 = vanderwaals1(273.15, 0.001, 1, b=23.71e-6, a=3.46e-3)

# 5. Mix: posizionali prima, poi per parola chiave
P5 = vanderwaals1(273.15, 0.001, 1, b=23.71e-6)

# 6. Tutti gli argomenti per parola chiave (ordine qualsiasi)
P6 = vanderwaals1(T=273.15, V=0.001, n=1, a=3.46e-3, b=23.71e-6)
P7 = vanderwaals1(n=1, V=0.001, T=273.15, a=3.46e-3, b=23.71e-6)
```

**Regole importanti:**
1. Gli argomenti **posizionali** (senza nome) devono venire **prima** di quelli per parola chiave
2. Gli argomenti con valori predefiniti devono essere dichiarati **dopo** quelli obbligatori nella definizione
3. Usare argomenti per parola chiave rende il codice pi√π leggibile: `vanderwaals1(T=300, V=0.002, n=1)` √® pi√π chiaro di `vanderwaals1(300, 0.002, 1)`

### Il Problema
Scrivi una funzione che calcola sia la circonferenza che l'area di un cerchio dato il raggio $r$.

**Formule:**
- Circonferenza: $L = 2\pi r$
- Area: $A = \pi r^2$

### Implementazione Versione 1: Import del Modulo Completo

```python
def circle1(r):
    """
    Calcola la circonferenza e l'area di un cerchio.
    
    INPUT:
        r - raggio del cerchio
    OUTPUT:
        L - circonferenza
        A - area
    """
    import math
    L = 2 * math.pi * r
    A = math.pi * r**2
    return L, A

# Test della funzione
help(circle1)  # Mostra la docstring

R = circle1(3)
print(f"Risultato: {R}")
print(f"Tipo: {type(R)}")  # <class 'tuple'>
```

**Output:**
```
Risultato: (18.84955592153876, 28.274333882308138)
Tipo: <class 'tuple'>
```

### Tuple: Strutture Dati Immutabili

Quando una funzione restituisce pi√π valori separati da virgola (`return L, A`), Python crea automaticamente una **tupla** - una sequenza ordinata e immutabile di valori.

```python
# Accesso agli elementi della tupla (indici partono da 0)
print(R[0])  # Prima elemento: circonferenza (18.849...)
print(R[1])  # Secondo elemento: area (28.274...)

# Tentativo di modifica: ERRORE!
R[1] = 6  # TypeError: 'tuple' object does not support item assignment
```

**Immutabilit√†:** Una volta creata, una tupla **non pu√≤ essere modificata**. Questo garantisce l'integrit√† dei dati.

### Unpacking: Assegnazione Multipla

Un modo elegante per lavorare con tuple √® l'"unpacking" (spacchettamento):

```python
# Assegna direttamente i valori a variabili separate
L, A = circle1(5)

# Accesso selettivo usando underscore per valori ignorati
_, A2 = circle1(4)  # Ignora la circonferenza, prendi solo l'area
```


### Esercizio 3: Triangolo Rettangolo

**Compito:** Scrivi una funzione Python che calcola le lunghezze dei cateti $b$ e $c$ di un triangolo rettangolo, dati l'ipotenusa $a$ e l'ampiezza di un angolo $\gamma$ (in radianti).

**Schema del triangolo:** (Vedi Lect1.pdf, slide 8-10 per immagini di riferimento sui concetti Python, anche se il triangolo specifico √® mostrato nel notebook)

```python
# Soluzione esempio
def triangle_sides(a, gamma):
    """
    Calcola i cateti di un triangolo rettangolo.

    OUTPUT:
        b - cateto adiacente all'angolo
```

## Riepilogo Concetti Chiave

### Funzioni in Python
- **Definizione:** `def nome_funzione(parametri):`
- **Indentazione obbligatoria** per definire i blocchi di codice
- **Docstring:** Documentazione accessibile con `help()`
- **Return multipli:** `return valore1, valore2` crea una tupla
- **Parametri predefiniti:** Permettono argomenti opzionali
- **Argomenti posizionali vs. per parola chiave:** Flessibilit√† nelle chiamate

### Moduli Python
- **Import completo:** `import math` ‚Üí uso: `math.pi`
- **Import con alias:** `import math as mt` ‚Üí uso: `mt.pi`
- **Import selettivo:** `from math import pi` ‚Üí uso: `pi`
- **Evitare:** `from math import *` (inquina il namespace)
- **Esplorazione:** `dir(modulo)` mostra contenuti, `help(funzione)` mostra documentazione

### Strutture Dati
- **Tuple:** Sequenze immutabili, create con `(val1, val2)` o `return val1, val2`
- **Accesso:** Indici da 0: `tupla[0]`, `tupla[1]`
- **Unpacking:** `var1, var2 = tupla`
- **Immutabilit√†:** Le tuple non possono essere modificate dopo la creazione
